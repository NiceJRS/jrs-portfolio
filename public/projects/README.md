# Project screenshots

Drop screenshot images here with these exact filenames. The site picks them up
automatically; until then each card shows a styled placeholder.

| Filename               | Project                          | Suggested shot                          |
| ---------------------- | -------------------------------- | --------------------------------------- |
| `parcel-dashboard.png` | Smart Parcel Management System   | Dashboard or parcel intake screen       |
